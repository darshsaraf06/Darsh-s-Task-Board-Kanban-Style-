Darsh's Task board(Kanban Task Management Application)

Deployed link: https://darsh-s-task-board-kanban-style.vercel.app/

A simple and interactive Kanban-style task management web application that allows users to organize tasks visually by moving them across different stages of progress.
This project was built as a single-page application and demonstrates core concepts of task management systems used in real-world productivity tools.

Features
	•	Create new tasks with titles and descriptions
	•	Organize tasks into task columns (To Do, In Progress, Done)
  •	Can add custome columns
	•	Drag and move tasks between columns
	•	Light / Dark theme toggle
	•	Theme preference is saved using localStorage
	•	Responsive and clean user interface
	•	Instant updates without page reload (local storage)

Technologies Used
	•	HTML
	•	CSS
	•	JavaScript
	•	React
	•	Vite (best with JS)
